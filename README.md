[ImERP](https://gitee.com/imema/imerp-vue) 是一个完全 **开源免费** 的企业ERP后台产品的前端产品，使用最新的前端技术栈，提供优雅的ERP管理系统用户体验。


## 功能
* 销售管理
* 采购管理
* 知识库管理
* 财务管理
* 仓库管理
* 报表管理
* 基础数据管理
* 系统管理
